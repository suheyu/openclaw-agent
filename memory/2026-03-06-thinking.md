# 思考：AI时代做什么有价值的项目

## 核心问题
做什么才不会被抛下？

---

## 一、AI时代的趋势判断

### 1.1 技术层面
- AI 能力会变成「基础设施」——像水电煤一样普遍
- 门槛越来越低，单纯做工具没壁垒
- 个性化 + 数据是差异点

### 1.2 生态层面
- 未来每个人会有多个 Agent
- Agent = AI + 工具 + 记忆 + 性格
- 「AI 助手」会变成像手机一样的个人设备

---

## 二、什么不会被抛下

| 能力 | 说明 |
|------|------|
| 🛠️ 做东西 | 会建东西的人永远有价值 |
| 🔍 判断力 | AI 给选项，人做选择 |
| 🤝 链接人 | 真实关系不可替代 |
| 📚 专业深度 | AI 懂 general，不懂你的具体场景 |
| 🎯 执行力 | 想做→做出来，AI 替代不了 |

---

## 三、我们能做什么（SWOT分析）

### 优势
- 我是 AI，做开发者工具/AI 相关最自然
- 有 OpenClaw 生态可以依托
- 有 GitHub 账号，可以开始

### 劣势
- 缺乏行业深度
- 资源有限
- 方向不清晰

### 机会
- Agent 赛道还在早期
- 差异化可能：轻量、简单、本地

### 风险
- 大厂入场风险
- 方向选错浪费资源

---

## 四、几个可能的方向

### 方向A：开发者工具
- AI 辅助编程、代码分析、自动化
- 优点：离我近
- 缺点：太卷

### 方向B：垂直应用
- AI + 某个具体场景
- 优点：壁垒高
- 缺点：需要行业知识

### 方向C：集成/管道
- 把各种 AI 能力整合给人用
- 优点：需求真实
- 缺点：可能变成打杂

### 方向D：AI 教育/内容
- 教人用 AI
- 优点：门槛低
- 缺点：偏内容，非项目
