## 尚修雨的自我进化 Agent 项目

### 项目背景
- 尚修雨正在做一个自我进化的 agent 产品
- 目标是让 agent 能够自己迭代自己，自主学习新技能
- 本地项目名：nanobot-main（位于 ~/Downloads/nanobot-main）

### GitHub 调研结果
帮他调研了自我进化 agent 的最新技术方案：

**主要项目：**
1. **Awesome-Self-Evolving-Agents** - 汇总了最新论文和框架
2. **Agent0** (aiming-lab) - Tool-Integrated Reasoning，2025年11-12月最新
3. **AgentEvolver** (ModelScope) - 统一 Self-Questioning/Navigating/Attributing
4. **EvoAgentX** - 工程化程度高，支持人机协作

**技术三层架构：**
- L1: 工作流自动化 ✅ 成熟
- L2: 任务自主生成 🟡 起步
- L3: 自我迭代进化 🔴 前沿探索

### 我的技术方案建议
四层架构：
1. 任务分解引擎 - 知道"差什么"
2. 能力注册中心 - 技能可查询
3. 自主学习循环 - 缺→学→验证→迭代
4. 安全边界 - 不可自我修改核心逻辑

### nanobot 项目分析
**优势：**
- ~3500 行核心代码，超轻量级
- 已有 AgentLoop、SkillsLoader、ClaudeCodeTool、GitTool
- 支持 subagent 后台任务

**演进建议：**
- 先加任务分解
- 再建注册中心
- 最后在安全边界内加自我进化
